# Plan: Create Proxy Route for Backend

## Goal
Add a proxy route in `projects/angular/server/app/routes.ts` that forwards requests to the backend running at `http://localhost:3000`.

## Changes Required

### File: `projects/angular/server/app/routes.ts`

1. **Import** `proxyRoute` from 'halide'
2. **Create** a proxy route with the following config:
   - `access: 'public'` — no auth required
   - `path: '/api/*'` — match all API routes
   - `methods: ['get', 'post', 'put', 'patch', 'delete']` — proxy all common HTTP methods
   - `target: 'http://localhost:3000'` — backend URL
   - `proxyPath: '/'` — strip `/api` prefix and forward to backend root
3. **Create** a new `proxyRoutes` array and **export** it alongside the existing `apiRoutes`

## Path Rewrite Behavior

```
Incoming:  /api/users/123
path:      /api/*
proxyPath: /
Result:    http://localhost:3000/users/123
```
